# Project Inspection Checklist

Inspect in this order:

1. `.titan/workflows/*.yaml`
2. `.titan/steps/**/*.py`
3. `.titan/operations/**/*.py`
4. `.titan/clients/**/*.py`
5. `.titan/services/**/*.py`

Look for:

1. extendable workflows
2. reusable step names
3. domain folders already present
4. existing integration code
